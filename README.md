# 📱 Mobile & Accessories Shop Management System

A desktop-based Mobile & Accessories Shop Management System developed using **C# Windows Forms** and **Microsoft SQL Server**. The application is designed to simplify retail shop operations by managing mobile phones, accessories inventory, customer records, sales transactions, and product information through an intuitive user interface.

<br>

## 🚀 Features

### 🔐 Authentication
- Secure Login System
- Authorized User Access

### 📦 Inventory Management
- Add New Mobile Phones
- Add Mobile Accessories
- Update Product Information
- Delete Product Records
- Search Products
- Stock Availability Tracking

### 🏷️ Product Management
- Mobile Brand & Model Management
- Accessories Inventory Management
- Product Categorization
- Product Pricing Management

### 🗄️ Database Integration
- Microsoft SQL Server Database
- Persistent Data Storage
- CRUD Operations
- Efficient Data Retrieval

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/tehzeeb-kousarr/mobile-shop-management-system.git
```

### Open the Project

```text
Open MobileShopManagementSystem.sln using Visual Studio
```

### Configure Database

1. Open SQL Server Management Studio (SSMS)
2. Create a new database
3. Import the provided SQL script
4. Update the connection string if required

### Run the Application

```text
Build → Start Debugging (F5)
```
